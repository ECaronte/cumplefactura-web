1) DEPENDENCIA Y SETUP GLOBAL
========================

1. Instala (o verifica que está instalada) la librería:
   - react-helmet-async

2. En el entrypoint principal (por ejemplo src/main.tsx o src/main.tsx + src/App.tsx, según la estructura de este repo):

   - Importa y envuelve la app con HelmetProvider:

   ```tsx
   import { HelmetProvider } from "react-helmet-async";

   ReactDOM.createRoot(document.getElementById("root")!).render(
     <React.StrictMode>
       <HelmetProvider>
         <App />
       </HelmetProvider>
     </React.StrictMode>
   );
````

* Si ya existe HelmetProvider, simplemente reutilízalo y NO dupliques proveedores.

3. En el index.html deja un <title> y <meta name="description"> genéricos de fallback, pero asume que en producción todo se controlará desde React con Helmet:

   * Title genérico: "CumpleFactura · Factura electrónica y VeriFactu para autónomos y gestorías"
   * Description genérica: "CumpleFactura te ayuda a cumplir con la factura electrónica, VeriFactu y firma con certificado digital sin complicaciones."
